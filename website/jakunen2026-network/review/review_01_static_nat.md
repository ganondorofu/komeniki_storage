# 模擬課題R-01: ルータ基礎総合（静的NAT含む）

## トポロジー

```
          内部ネットワーク           外部ネットワーク
  172.20.0.0/24                     100.64.0.0/30
  Lo0         Fa0/1   Fa0/0                    [R1/ISP]
  172.20.0.1  ----   [R2]   .1 ---- .2   100.64.0.2
              .1
```

| ノード | IF | IPアドレス | 備考 |
|--------|-----|-----------|------|
| R1 | Fa0/0 | 100.64.0.2/30 | ISP側 |
| R2 | Fa0/0 | 100.64.0.1/30 | 外部（NAT outside） |
| R2 | Fa0/1 | 10.20.30.1/24 | 内部LAN（NAT inside） |
| R2 | Lo0 | 172.20.0.1/32 | PC代わり |

NAT変換: `172.20.0.1` → `100.64.0.10`（静的）

---

## 要件

### 1. ターミナル環境設定（R1・R2 両方）
全ルータに標準のターミナル環境設定を行う。

### 2. インタフェース設定（R1・R2）
上記のIPアドレスを全インタフェースに設定し、up状態にする。

### 3. ルーティング
- R2: デフォルトルートを R1（100.64.0.2）向けに設定する
- R1: `100.64.0.10/32` への経路を R2（100.64.0.1）向けに設定する

### 4. 静的NAT（R2）
- Lo0のIPアドレス（172.20.0.1）を外部アドレス（100.64.0.10）に静的NATで変換する
- 内部・外部インタフェースを適切に設定する

---

## 達成確認

- R1から `ping 100.64.0.10` が通ること
- R2で `show ip nat translations` に変換エントリが表示されること
