## 1. まずは動かしてみよう

プログラミングの第一歩は「Hello, World!」です。以下のコードを書いて実行してみましょう。

```python
print("Hello, World!")
```

**実行結果:**
```
Hello, World!
```

### 解説
- `print()` は画面に文字を表示する命令です。
- `""` で囲まれた部分が表示されます。

:::tip 試してみよう
`"Hello, World!"` を `"こんにちは！"` に変えて実行してみよう！
:::

## 2. 計算してみよう

Pythonは計算機としても使えます。

```python
print(1 + 2)
print(10 - 3)
print(4 * 5)
print(20 / 4)
print(17 // 5)
print(17 % 5)
```

**実行結果:**
```
3
7
20
5.0
3
2
```

### 計算の記号
| 記号 | 意味 | 例 |
|------|------|------|
| `+` | 足し算 | `1 + 2` → 3 |
| `-` | 引き算 | `10 - 3` → 7 |
| `*` | 掛け算 | `4 * 5` → 20 |
| `/` | 割り算 | `20 / 4` → 5.0 |
| `//` | 割り算（整数部分のみ） | `17 // 5` → 3 |
| `%` | 割り算の余り | `17 % 5` → 2 |

:::info 余りのある割り算
`17 ÷ 5 = 3 あまり 2` のとき：
- `17 // 5` → 3（商）
- `17 % 5` → 2（余り）
:::

:::tip 試してみよう
好きな計算をしてみよう！ `123 * 456` とか。
:::

## 3. 変数を使ってみよう

変数は「データを入れておく箱」です。名前をつけて値を保存できます。

```python
x = 10
print(x)
```

**実行結果:**
```
10
```

### 変数を使った計算

```python
a = 5
b = 3
c = a + b
print(c)
```

**実行結果:**
```
8
```

### 文字も入れられる

```python
name = "太郎"
print(name)
```

**実行結果:**
```
太郎
```

:::info ポイント
- `=` は「右の値を左の変数に入れる」という意味です。
- 数学の「等しい」とは違うので注意！
:::

## 4. データの種類（型）について詳しく学ぼう

プログラミングで最も重要な概念の一つが「型（かた）」です。これをしっかり理解しないと、後で必ずつまずきます。

### 型とは何か？

コンピュータにとって、`10` と `"10"` は**まったく別のもの**です。

- `10` → 数値（計算できる）
- `"10"` → 文字列（文字として扱う。計算できない）

人間から見ると同じ「10」ですが、コンピュータは区別しています。この違いを「型」と呼びます。

### なぜ型が必要なの？

実際に試してみましょう。

```python
# 数値の足し算
a = 10
b = 20
print(a + b)
```

**実行結果:**
```
30
```

これは期待通り。では、文字列ではどうでしょう？

```python
# 文字列の足し算
a = "10"
b = "20"
print(a + b)
```

**実行結果:**
```
1020
```

「30」ではなく「1020」になりました！ 文字列の `+` は「連結（くっつける）」という意味になるからです。

### 型の種類

Pythonには主に4つの基本型があります。

| 型 | 名前 | 説明 | 例 |
|------|------|------|------|
| `int` | 整数型 | 小数点のない数字 | `10`, `-5`, `0`, `12345` |
| `float` | 浮動小数点型 | 小数点のある数字 | `3.14`, `-0.5`, `2.0` |
| `str` | 文字列型 | 文字の並び | `"こんにちは"`, `'Hello'`, `"123"` |
| `bool` | ブール型 | 真か偽か | `True`, `False` |

### 型を確認する方法

`type()` 関数で確認できます。

```python
print(type(10))       # <class 'int'>
print(type(3.14))     # <class 'float'>
print(type("Hello"))  # <class 'str'>
print(type(True))     # <class 'bool'>
```

### 文字列の見分け方

文字列は必ず `""` か `''` で囲まれています。

```python
x = 123      # これは数値（int）
y = "123"    # これは文字列（str）

print(type(x))  # <class 'int'>
print(type(y))  # <class 'str'>
```

:::tip ポイント
`""` で囲まれていれば、中身が数字でも**文字列**です！
:::

### 型を変換する（キャスト）

型は変換できます。これを「キャスト」と呼びます。

| 変換関数 | 役割 | 例 |
|------|------|------|
| `int()` | 整数に変換 | `int("123")` → `123` |
| `float()` | 小数に変換 | `float("3.14")` → `3.14` |
| `str()` | 文字列に変換 | `str(123)` → `"123"` |

#### 例：文字列を数値に変換

```python
age_str = "16"           # 文字列の「16」
age_int = int(age_str)   # 数値の 16 に変換
print(age_int + 1)       # 17（計算できる！）
```

#### 例：数値を文字列に変換

```python
score = 95
message = "あなたの点数は" + str(score) + "点です"
print(message)
```

**実行結果:**
```
あなたの点数は95点です
```

:::warning 変換できない場合はエラー
```python
int("こんにちは")  # エラー！ 数字じゃないので変換できない
int("3.14")        # エラー！ 小数は int() で変換できない（float()を使う）
```
:::

### 練習：型を意識してみよう

以下のコードの出力を予想してみてください。

```python
a = "3"
b = "5"
print(a + b)        # ①

a = 3
b = 5
print(a + b)        # ②

a = "3"
b = 5
print(int(a) + b)   # ③
```

<details>
<summary>答えを見る</summary>

- ① `35`（文字列の連結）
- ② `8`（数値の足し算）
- ③ `8`（文字列を数値に変換してから足し算）

</details>

## 5. 文字と数字を組み合わせる

変数を使って、文字と数字を組み合わせて表示できます。

### 方法1: カンマで区切る

```python
age = 16
print("私は", age, "歳です")
```

**実行結果:**
```
私は 16 歳です
```

### 方法2: f文字列（おすすめ）

```python
age = 16
print(f"私は{age}歳です")
```

**実行結果:**
```
私は16歳です
```

:::tip f文字列
`f"..."` の中で `{変数名}` と書くと、変数の値が埋め込まれます。便利！
:::

## 6. 入力を受け取ってみよう

ユーザーからの入力を受け取ることができます。

```python
name = input("あなたの名前は？ ")
print(f"こんにちは、{name}さん！")
```

**実行例:**
```
あなたの名前は？ 太郎
こんにちは、太郎さん！
```

### 数字を入力する場合

`input()` は文字として受け取るので、計算に使うには `int()` で数字に変換します。

```python
age = input("あなたの年齢は？ ")
age = int(age)
print(f"来年は{age + 1}歳ですね")
```

**実行例:**
```
あなたの年齢は？ 16
来年は17歳ですね
```

:::warning 注意
`int()` を忘れると、文字として扱われてエラーになります。
:::

## 7. コメントを書く

コードにメモを残すことができます。`#` 以降はプログラムに影響しません。

```python
# これはコメントです
print("Hello")  # ここにもコメント書ける
```

**実行結果:**
```
Hello
```
