# 新入部員の皆さんへ

ようこそ、STEM研究部へ！ ここでは、高校生向けのSTEM（Science, Technology, Engineering, Mathematics）ドキュメントを集めています。プログラミングや科学技術を深く学び、プロジェクト制作に挑戦しましょう。

## 部について

STEM研究部は、高校生向けのSTEM教育を目的とした部活動です。プログラミング、AI、IoT、データサイエンスなどを通じて、未来の技術者を育成することを目指しています。

### 活動内容
- **プログラミング学習**: HTML、JavaScript、C、C#などを学びます。
- **プロジェクト制作**: Webアプリ、ゲーム、ロボット作成。
- **研究と発表**: 科学的なテーマを研究し、発表会で共有。
- **コンテスト参加**: 若年者モノづくり競技大会、技能五輪のような大会やU22プログラミングコンテストなどに参加しています。


何か質問があれば、先生や先輩に聞いてね！ 一緒にSTEMの世界を探求しよう！
